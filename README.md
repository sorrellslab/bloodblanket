## Overview
Design files and code for running the blood blanket (BB) behavior assay that measures mosquito arousal and engorgement in response to optogenetic and heat stimulation

## Repository Structure
```text
├── Aluminum_plate_templates/               # Templates to create the aluminum blood holder
├── Arduino/                                # Arduino program for light stimulus delivery
├── BB_thermocycler_program.x50prog/        # Thermocycler program for heat stimulus delivery
├── models/                                 # Sleap models for top-down analysis for predicting on BB videos
